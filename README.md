# PaPaPhone

Голосовой телефон для людей с потерей зрения. Полностью офлайн, управление только голосом.

Работает на Orange Pi Zero 2W + SIM7600G-H 4G HAT. Можно адаптировать под любой Linux.

## Как это работает

Дедушка говорит **"телефон позвони маме"** -- одной фразой. Система распознаёт речь, находит контакт по имени (с учётом склонений: "маме" = "мама"), переспрашивает подтверждение и звонит. Всё офлайн, без интернета.

Архитектура -- конечный автомат (FSM) с пятью состояниями. Входящие звонки и SMS обнаруживаются автоматически фоновыми потоками.

## Голосовые команды

После wake-фразы **"телефон"** (или в одной фразе с ней):

### Звонки

| Команда | Пример | Что происходит |
|---------|--------|----------------|
| Позвонить | "позвони маме", "позвонить сыну" | Поиск по имени/прозвищу (морфология) -> подтверждение -> звонок |
| Набрать номер | "набери номер 89001234567" | Подтверждение -> набор |
| Ответить | "ответь", "ответить" | Принять входящий |
| Положить трубку | "положи трубку", "положить трубку" | Завершить звонок |

### SMS

| Команда | Пример | Что происходит |
|---------|--------|----------------|
| Сообщения | "сообщения" | Все SMS по одному: дальше / позвонить / хватит |
| Новые сообщения | "новые сообщения" | Только непрочитанные |
| *(автоматически)* | | Новые SMS озвучиваются при поступлении. Ночью копятся, утром зачитываются |

### Контакты

| Команда | Пример | Что происходит |
|---------|--------|----------------|
| Найти контакт | "найди контакт маму" | Поиск -> позвонить / прозвище / переименовать |
| Добавить контакт | "добавь контакт" | Имя -> номер по цифрам -> прозвище (необязательно) |
| Прозвище | "прозвище маме" | Добавить альтернативное имя к контакту |
| Список контактов | "список контактов" | Зачитать все с пометкой экстренных |

### Информация

| Команда | Пример |
|---------|--------|
| Время | "который час", "сколько времени" |
| Адрес | "адрес" |
| Команды | "команды" |

### Экстренные

| Команда | Пример | Что происходит |
|---------|--------|----------------|
| Экстренный вызов | "спасите", "скорая", "мне плохо", "помогите", "нужна помощь" | Звонок 112. Тишина = звонить (не отменять). **SMS всем экстренным контактам** с текстом "Экстренный вызов! Вызвана помощь. Адрес: ..." |

Экстренные контакты помечаются через секретное меню и хранятся в БД. При вызове 112 каждый экстренный контакт получает SMS с адресом автоматически -- до набора номера.

### Управление

| Команда | Пример | Что происходит |
|---------|--------|----------------|
| Отмена | "стоп", "отмена" | Прервать текущее действие (с подтверждением) |

### Секретное меню

Вход по фразе **"секретное меню"**, далее называешь имя контакта:

- **"удалить"** -- удалить контакт (с подтверждением)
- **"экстренный"** -- пометить как экстренный (получит SMS при вызове 112)
- **"убрать из экстренных"** -- снять пометку

## Входящие звонки

Система обнаруживает входящий через AT+CLCC каждые 2 секунды. Определяет кто звонит по контактам. Спрашивает "Входящий от Мама. Ответить?" Молчание = сброс (безопасный дефолт).

## Конечный автомат (FSM)

```
IDLE -> LISTENING -> EXECUTING -> IDLE
  |        |            |
  |        |            +-> IN_CALL -> IDLE
  |        |
  +-> INCOMING -> IDLE
```

- **IDLE** -- ждёт wake-фразу, проверяет входящие звонки и SMS
- **LISTENING** -- wake получен, ждёт команду (12 с)
- **EXECUTING** -- сценарий выполняется
- **IN_CALL** -- активный звонок, слушает "положи трубку"
- **INCOMING** -- обработка входящего звонка

Команда может быть в одной фразе с wake: "телефон позвони маме" -- без паузы.

## Технологии

| Компонент | Технология |
|-----------|------------|
| Распознавание речи | [Vosk](https://alphacephei.com/vosk/) (офлайн) |
| Синтез речи | [Piper TTS](https://github.com/rhasspy/piper) (офлайн, нейросеть) |
| Модем | SIM7600G-H по AT-командам через USB (pyserial) |
| Контакты | SQLite + pymorphy3 (морфология, склонения, прозвища) |
| Команды | YAML-словарь + резолв имён через БД |

## Оборудование

- **Orange Pi Zero 2W** (4 GB RAM) или любой Linux ARM/x86
- **SIM7600G-H 4G HAT** -- подключение через USB к Orange Pi (определяется как `/dev/ttyUSB0`)
- **USB-микрофон**
- **Динамик** (USB-аудио или 3.5 мм)

SIM7600 подключается к Orange Pi Zero 2W через USB -- модем определяется как последовательный порт `/dev/ttyUSB0`. Если порт другой, задайте в `data/.env`.

## Установка

```bash
git clone https://github.com/Volk-Arch/PaPaPhone.git
cd PaPaPhone
bash deploy.sh
```

Скрипт установит зависимости, скачает модели Vosk (45 MB) и Piper, настроит конфигурацию (адрес, порт модема), создаст systemd-сервис с автозагрузкой.

### Ручная установка

```bash
sudo apt install python3 python3-venv portaudio19-dev
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# Модель Vosk (русская, 45 MB)
mkdir -p models
wget https://alphacephei.com/vosk/models/vosk-model-small-ru-0.22.zip
unzip vosk-model-small-ru-0.22.zip -d models/

# Модель Piper TTS
BASE=https://huggingface.co/rhasspy/piper-voices/resolve/main/ru/ru_RU/ruslan/medium
wget -P models/ $BASE/ru_RU-ruslan-medium.onnx
wget -P models/ $BASE/ru_RU-ruslan-medium.onnx.json

# Конфигурация
cp data/.env.example data/.env
nano data/.env
```

## Запуск

```bash
# Продакшн (Orange Pi + модем)
python -m src.main

# Отладка без модема
python -m src.main --demo

# Без wake-фразы (быстрее тестировать)
python -m src.main --demo --no-wake
```

### Управление сервисом (после deploy.sh)

```bash
sudo systemctl start papaphone       # запустить
sudo systemctl stop papaphone        # остановить
sudo systemctl restart papaphone     # перезагрузить
sudo systemctl status papaphone      # статус
journalctl -u papaphone -f           # логи в реальном времени
journalctl -u papaphone --since today # логи за сегодня
```

Сервис запускается автоматически при включении устройства. При ошибке перезапускается через 10 секунд.

## Конфигурация

Файл `data/.env` (создаётся при `deploy.sh` или вручную из `data/.env.example`):

| Переменная | По умолчанию | Описание |
|------------|--------------|----------|
| `PAPAPHONE_VOSK_MODEL` | `vosk-model-small-ru-0.22` | Папка модели в `models/` |
| `PAPAPHONE_MODEM_PORT` | `/dev/ttyUSB0` | Порт SIM7600 (USB) |
| `PAPAPHONE_WAKE_PHRASES` | `телефон,папафон` | Wake-фразы через запятую |
| `PAPAPHONE_EMERGENCY_NUMBER` | `112` | Номер экстренного вызова |
| `PAPAPHONE_HOME_ADDRESS` | *(не задан)* | Домашний адрес (команда "адрес" + SMS при экстренном вызове) |
| `PAPAPHONE_AUDIO_INPUT_DEVICE` | *(системный)* | Индекс ALSA микрофона |
| `PAPAPHONE_AUDIO_OUTPUT_DEVICE` | *(системный)* | Индекс ALSA динамика |

Экстренные контакты настраиваются голосом через секретное меню и хранятся в БД.
