# PaPaPhone: тест-сценарии

## Запуск

```bash
python -m src.main --demo           # с wake-фразой
python -m src.main --demo --no-wake # без wake (быстрее)
python -m src.main                  # продакшн
```

`Enter` в демо = собеседник повесил трубку.

---

## Группа 1: Wake + single utterance

| # | Говорим | Ожидаем | FSM |
|---|---------|---------|-----|
| 1.1 | "телефон" | "Слушаю." | IDLE -> LISTENING |
| 1.2 | "телефон позвони маме" | Поиск контакта, подтверждение | IDLE -> EXECUTING |
| 1.3 | "телефон который час" | Время | IDLE -> EXECUTING |
| 1.4 | "телефон пожалуйста позвони маме" | Мусор отфильтрован, slot "маме" | IDLE -> EXECUTING |
| 1.5 | "привет" (без wake) | Молчит | IDLE |
| 1.6 | "телефон" -> (тишина 12с) | "Слушаю." -> таймаут | LISTENING -> IDLE |

## Группа 3: Морфология и прозвища

Предварительно: контакт "Игорь" с прозвищем "сын".

| # | Говорим | Ожидаем |
|---|---------|---------|
| 3.1 | "телефон позвони игорю" | Найден: "игорю" -> БД -> "Игорь" |
| 3.2 | "телефон позвони сыну" | Найден через алиас: "сыну" -> морфология -> "сын" -> алиас Игоря |
| 3.3 | "телефон позвони маме" | Найден (если есть контакт Мама) |
| 3.4 | "телефон позвони папе" | Найден (если есть) |

## Настройка

```bash
nano data/.env   # адрес, порт модема, wake-фразы
```

Экстренные контакты -- через секретное меню (голосом), хранятся в БД.
