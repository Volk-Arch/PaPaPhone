# PaPaPhone: тест-сценарии

## Запуск

```bash
# Тестирование (с wake-фразой, как в продакшне)
python -m src.main --demo

# Orange Pi -- продакшн
python -m src.main
```

Горячие клавиши в демо: `Enter` = собеседник повесил трубку.

---

## Группа 1: Wake + single utterance

| # | Говорим | Ожидаем | FSM |
|---|---------|---------|-----|
| 1.1 | "телефон" | "Слушаю." | IDLE -> LISTENING |
| 1.2 | "телефон позвони тесту" | Поиск контакта, подтверждение | IDLE -> EXECUTING |
| 1.3 | "папафон который час" | Время | IDLE -> EXECUTING |
| 1.4 | "телефон пожалуйста позвони тесту" | Strip wake + стоп -> slot "тесту" | IDLE -> EXECUTING |
| 1.5 | "телефон ну давай позвони маме" | Strip мусор -> "позвони маме" | IDLE -> EXECUTING |
| 1.6 | "привет" (без wake) | Молчит | IDLE |
| 1.7 | (тишина 3с) | Молчит | IDLE |
| 1.8 | "телефон" -> (тишина 12с) | "Слушаю." -> таймаут | LISTENING -> IDLE |
